CO685 (The Mathematics of Public Key Cryptography) - Course Project
This document contains information about the course project titled "Point Multiplication on Elliptic Curves", submitted for the CO685 course for the Fall 2015 term. The project involves implementing an arbitrary-length integer handling class, an elliptic curve point operations class and an executable driver program.

Implemented classes and algorithms:
  • BigInteger class - to store and perform arithmetic operations on arbitrarily large signed integers.
  • ECPO class - to validate elliptic curves and points on those curves, and perform elliptic curve point multiplication and related algorithms.
    Point multiplication algorithms implemented and analysed:
    • Brute-force point multiplication (repeated point addition)
    • Double-and-add
    • window-NAF
    Note:
    • Sliding-window-NAF was deliberately omitted - see Design Details.
    • The Montgomery ladder method from Wikipedia appears to be incorrect - and the theory in "Guide to Elliptic Curve Cryptography" is beyond what we have studied in the course. This has been omitted as well.
